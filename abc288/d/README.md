- 以下、添字は0-indexedで考える。
- k := l mod K として、k != 0の時、全体をkだけシフトし、l := l - k, r := r - kとした問題を考えれば良い。
- [l, r]を左から順に0になるようにK個範囲加算していく操作を繰り返す。
- この操作を残りK-1個になるまで繰り返して、残った x = y, ..., r における値が全部0になっていれば良い。(y:=r-(K-1))
- 実験すると、xにおける値はP-Qであることが分かる。ここで、
  - Pはl<=i<=xのうちi mod Kがxと同じA[i]の和
  - Qはl<=i<=sのうちi mod Kがsと同じA[i]の和 (s:=r-(K-1))
- 上記の和を、K個飛ばしの累積和で計算すれば間に合う。
